Images

Opt your images into responsive behavior (so they never become larger than their parent elements) and add lightweight styles to them—all via classes.

Contents

  • Contents
  • Responsive images
  • Image shapes
  • Aligning images

Responsive images

Images in Bootstrap are made responsive with .img-fluid. max-width: 100%; and height: auto; are applied to the image so that it scales with the parent element.

SVG images and IE 9-10

In Internet Explorer 9-10, SVG images with .img-fluid are disproportionately sized. To fix this, add width: 100% \9; where necessary. This fix improperly sizes other image formats, so Bootstrap doesn’t apply it automatically.

Image shapes

Add classes to an <img> element to easily style images in any project.

Aligning images

Align images with the helper float classes or text alignment classes. block-level images can be centered using the .m-x-auto margin utility class.
